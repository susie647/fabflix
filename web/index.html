'A_Z': All strings that start with 'A', then follow by a single character, and end with 'Z'. E.g. 'ABZ' and 'A2Z'. However, 'AKKZ' does not satisfy the condition. 'ABC%': All strings that start with 'ABC'. E.g. 'ABCD' and 'ABCABC'. '%XYZ': All strings that end with 'XYZ'. E.g. 'WXYZ' and 'ZZXYZ'. '%AN%': All strings that contain the pattern 'AN' anywhere. E.g. 'LOS ANGELES' and 'SAN FRANCISCO'.
